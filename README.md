# sayuprc/date-type

Simple DateTime-based date type.

## Requirements

|name|version|
|---|---|
|PHP|^8.4|

## Installation

```bash
composer require sayuprc/date-type
```

## Usage

```php
<?php

use DateType\ImmutableDate;

// A date is generated with the time in the state 00:00:00.000000
$date = new ImmutableDate();
```
